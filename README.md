In-Memory Search Engine in C

A simple search engine built in C that stores documents in memory and retrieves results based on keyword relevance.

Features
Add and store documents in memory
Prevent duplicate entries
Keyword-based search (strstr)
Frequency-based ranking
Case-insensitive search
Menu-driven interface

Concepts Used
Arrays & Strings
Functions (modular coding)
String handling (fgets, strcmp, strstr)
Input validation (strtol)
Sorting (Bubble Sort)
Linear Search (O(n))

How to Run
gcc main.c -o search
./search
